╌╌╌ TextGrid Labeler User Guide ╌╌╌

[File Operations]
  • Ctrl+N          New TextGrid (from template or default words/phones tiers)
  • Ctrl+Shift+N    New from Current (clone current file's tier structure, clear labels)
  • Ctrl+O          Open TextGrid (auto-loads WAV from same / parent / wav/ directory)
  • Ctrl+W          Manually open WAV audio file
  • Ctrl+S          Save TextGrid
  • Ctrl+Shift+S    Save TextGrid as new file
  • Recent Files    Submenu listing recently opened files; click to reopen
  • Project menu    Lists all .TextGrid files in the current file's directory for quick switching

[Annotation Editing]
  • Click & drag a red boundary line         Adjust boundary position
  • Double-click a red boundary line         Remove boundary (merges intervals, keeps left label)
  • Double-click inside an interval          Add new boundary (prompts for label)
  • Double-click a label in tier bar         Change interval label
  • Click a label in tier bar                Select interval (highlight + list select + status info)
  • Right-click a label in tier bar          Context menu: change label, add boundary at center, remove interval
  • Right-click an interval in waveform      Play audio for that segment
  • [ (bracket-open key on waveform)         Quick-add boundary at cursor position (same label as current interval)
  • ] (bracket-close key on waveform)        Quick-delete current interval (merges with previous)

[Search & Navigation]
  • Type keyword + press Enter    Search interval labels (substring match)
  • ◀ / ▶ buttons                 Jump to previous / next search result
  • Click intervals list item    Jump to and highlight that interval
  • Count label                   Shows "N/M" for current search position

[Intervals List (right panel)]
  • Three columns: Label, Start (s), Dur (ms)
  • Automatically filters when search is active
  • Click any item to jump to and highlight that interval
  • Selection synced with search navigation

[View Operations]
  • Mouse wheel              Horizontal scroll
  • Ctrl + mouse wheel       Zoom in / out (centered on cursor)
  • Bottom scroll bar        Horizontal scroll
  • dB ruler on right        Shows audio amplitude levels (0 to -48 dB)

[Undo & Redo]
  • Ctrl+Z    Undo last edit
  • Ctrl+Y    Redo last undone edit
  • History   Up to 100 steps; cleared when opening a new file

[Tier Switching]
  • Dropdown in toolbar   Switch the current editing tier

[FAQ]
  • WAV auto-load priority for a TextGrid at path/to/file.TextGrid:
    1. path/to/file.wav
    2. path/file.wav
    3. path/wav/file.wav, path/wavs/file.wav, path/WAV/file.wav, path/WAVS/file.wav
  • The status bar shows current file and operation feedback
  • Closing the window or opening another file prompts to save unsaved changes
  • Template.TextGrid: if present beside the executable, used as tier template for New TextGrid
